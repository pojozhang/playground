# [2的幂](https://leetcode-cn.com/problems/power-of-two/description/)

给定一个整数，写一个函数来判断它是否是 2 的幂次方。

**示例 1:**

```
输入: 1
输出: true
```

**示例 2:**

```
输入: 16
输出: true
```

**示例 3:**

```
输入: 218
输出: false
```
